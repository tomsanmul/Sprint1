Instruccions per a l'execució de cada part del mòdul 5.
-----------------------------------------------------------

PRIMER: instalar el mòdul ZIP per a l'exercici 1.3 desde la terminal:
npm install adm-zip


SEGON: Cada exercici està comentat amb '/*' d'inici i '*/' com a final.
S'ha descomentar cada Exercici, correlativament, i executar-lo d'un en un.
Un cop executat, s'ha de tornar a comentar amb les mateixes '/*'  '*/'. 
Exemple:

//Nivell 1
//Exercici 1
//Crea una funció que, en executar-la, escrigui una frase en un fitxer.
/*
function {....}

Executa funció;
*/
